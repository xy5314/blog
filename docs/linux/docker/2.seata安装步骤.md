# docker 安装 seata

:::success 
docker 运行命令
:::
```shell
docker run -d \
 --name seata \
 --network develop_network \
 --privileged=true \
 -p 38091:38091 \
 -p 37091:7091 \
 -e SEATA_IP=10.10.100.1 \
 -e SEATA_PORT=38091 \
 -v /data/docker/seata/resources:/seata-server/resources \
 -v /etc/localtime:/etc/localtime:ro \
 --restart=always \
 seataio/seata-server:1.6.1
```
