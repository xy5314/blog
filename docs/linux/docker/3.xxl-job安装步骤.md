# docker 安装 xxl-job

:::success 
docker 运行命令
:::
```shell
docker run -d \
  --name xxl-job \
  --network develop_network \
  -p 30180:8080 \
  -e PARAMS="--spring.datasource.url=jdbc:mysql://mysql:3306/xxl_job?useUnicode=true&characterEncoding=UTF-8&autoReconnect=true&serverTimezone=Asia/Shanghai --spring.datasource.username=root --spring.datasource.password=123456 --xxl.job.accessToken=Token --server.servlet.context-path=/xxl-job-admin" \
  -v /etc/localtime:/etc/localtime:ro \
  --restart=always \
  xuxueli/xxl-job-admin:2.4.1
```
