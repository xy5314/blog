# vscode 
:::note
配置code runner的c++运行环境
:::

```json showLineNumbers title=".vscode/settings.json"
 "code-runner.executorMap": {
     "c": "cd $dir &&  gcc $fileName -o $fileNameWithoutExt && $dir$fileNameWithoutExt && rm $dir$fileNameWithoutExt",
        "cpp": "cd $dir && g++ $fileName -o $fileNameWithoutExt && $dir$fileNameWithoutExt && rm $dir$fileNameWithoutExt",
 }
```

# zsh iTerm2光标
:::note
行内光标
:::

```json showLineNumbers

到行首：ctrl + a
 
到行尾：ctrl + e
 
前进后退：ctrl + f/b (相当于左右方向键)
 
上一条命令：ctrl + p
 
搜索命令历史：ctrl + r
 
删除当前光标的字符：ctrl + d
 
删除光标之前的字符：ctrl + h
 
删除光标之前的单词：ctrl + w
 
删除光标到文本末尾：ctrl + k

清除当前行：ctrl + u
 
交换光标处文本：ctrl + t
 
清屏1：command + r
 
清屏2：ctrl + l
 
```
