# adb的连接方式

- 手机打开开发者模式
- 数据线连接上手机
- 获取手机列表


# 常用命令

```bash   showLineNumbers
# 获取手机列表
adb devices

# 转发到5555(默认端口)
adb tcpip 5555

# 查询手机的IP地址
adb shell ip addr show wlan0

# 模拟点击
adb shell input tap {x} {y}

# 获取手机电池状态 level就是当前电量
adb shell dumpsys battery
```
