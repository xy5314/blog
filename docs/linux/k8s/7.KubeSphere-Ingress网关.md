# KubeSphere-Ingress网关


```shell 

kind: Ingress
apiVersion: networking.k8s.io/v1
metadata:
  name: oa-api
  namespace: devops
  annotations:
    kubesphere.io/creator: admin
    nginx.ingress.kubernetes.io/rewrite-target: /api/$1
    nginx.ingress.kubernetes.io/use-regex: 'true'
spec:
  tls:
    - hosts:
        - oa-api.test.17111321.xyz
      secretName: lianyun-dev-tls-tv66d
  rules:
    - host: oa-api.test.17111321.xyz
      http:
        paths:
          - path: /(.*)
            pathType: Prefix
            backend:
              service:
                name: oa-api
                port:
                  number: 80

```
